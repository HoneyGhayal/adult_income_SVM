# Adult Income Prediction using SVM

##  Project Description
This project uses Machine Learning (Support Vector Machine) to predict whether a person earns more than $50K per year based on demographic data.

##  Algorithm Used
- Support Vector Machine (SVM)

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

##  Steps Involved
- Data cleaning
- Feature encoding
- Train-test split
- Feature scaling
- Model training using SVM

##  Goal
To classify income level based on census data with good accuracy.

##  Output
Model predicts income category: `<=50K` or `>50K`
